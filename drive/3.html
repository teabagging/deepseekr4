Let's see:The positive terms are: 1 + 1/2 + 1/3 + 1/4 + ... + 1/nThe negative terms are: -1/4 - 1/5 - 1/6 - ... -1/(n + 3)So, when we subtract, the terms from 1/4 to 1/n will cancel out in both positive and negative parts.Therefore, the remaining terms are:Positive: 1 + 1/2 + 1/3Negative: -1/(n + 1) - 1/(n + 2) - 1/(n + 3)So, the sum becomes:(1 + 1/2 + 1/3) - (1/(n + 1) + 1/(n + 2) + 1/(n + 3))Calculating the positive part:1 + 1/2 + 1/3 = (6/6) + (3/6) + (2/6) = 11/6So, T_n = (2/3)(11/6 - (1/(n + 1) + 1/(n + 2) + 1/(n + 3)))Simplify:T_n = (2/3)(11/6) - (2/3)(1/(n + 1) + 1/(n + 2) + 1/(n + 3))Calculating (2/3)(11/6):(2/3)(11/6) = 22/18 = 11/9So,T_n = 11/9 - (2/3)(1/(n + 1) + 1/(n + 2) + 1/(n + 3))Since all the terms 1/(n + 1), 1/(n + 2), 1/(n + 3) are positive, the subtracted part is positive. Therefore,T_n = 11/9 - [positive term] < 11/9Hence, T_n < 11/9.Let me verify this with n=1,2,3 to see if it makes sense.For n=1:S_1 = 1(1 + 3)/2 = 2. So, 1/S_1 = 1/2.T_1 = 1/2 = 0.5 < 11/9 ≈ 1.222. True.For n=2:S_1 = 2, S_2 = 2 + 3 = 5. So, 1/S_1 + 1/S_2 = 1/2 + 1/5 = 7/10 = 0.7 < 11/9. True.For n=3:S_3 = 3(3 + 3)/2 = 9. So, 1/S_3 = 1/9.T_3 = 1/2 + 1/5 + 1/9 = (45 + 18 + 10)/90 = 73/90 ≈ 0.811 < 11/9. True.And as n increases, the subtracted terms get smaller, so T_n approaches 11/9 from below. Therefore, T_n is always less than 11/9.So, part (2) is proven.
